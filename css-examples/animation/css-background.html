background-origin属性指定了背景图像的位置区域：

  • padding-box（默认值），从padding的外边缘开始显示背景图片
  • border-box，从border的外边缘开始显示背景图片
  • content-box，从content的外边缘开始显示背景图片

background-clip定义背景图像的剪裁区域：

  • padding-box，出现在padding区域之外的背景被裁剪
  • border-box（默认值），出现在border区域之外的背景被裁剪
  • content-box，content区域之外的背景被裁剪
